Purpose

Provide a comprehensive and standalone way to perform Static Code Analysis on Apex classes of a salesforce org.



Org Selection: View registered orgs and API Version for which SCA is to be performed.

Apex Class Selection:List of Apex classes.

PMD Rule Selection:Dynamically fetched list of Apex Rules along with Priority and Deprecation Status.

SCA Report:Displays the Report highlighting key parameters like frequency, type and distribution and details of violations.
  • Salesforce OAuth Web server Flow
  • Usage of Salesforce Metadata API for apex class retrieval
  • Spring Data JPA with Complex table relations
  • JWT based Authentication in Frontend as well as Backend
  • Prime Ng library for variety of View components
  • Apex PMD Java API for SCA
